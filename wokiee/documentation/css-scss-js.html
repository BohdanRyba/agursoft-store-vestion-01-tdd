Css and Scss

All css and scss files located at css folder.
To create your own theme and change styles you must use files in - scss/style.scss

In css/theme.scss - there are variables with which you can easily customize the theme.

For example, change the default text color: 

					$default_text_color:              					#777777;
					$default_color:                  					#2879fe;

JS

All JS files located at js folder and external.

  • js/main.js - Theme main JS file.
  • external/ - vendor plugins JS files

There are also separately connected blocks:

  • separate-include/blog (*for blog pages)
  • separate-include/listing (*for listing pages)
  • separate-include/main-slick-slider (* If you want to use "Slick slider" instead "Revolution Slider". Examples)
  • separate-include/portfolio (*for portfolio pages)
  • separate-include/single-product (*for single product pages)
